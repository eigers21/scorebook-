# 野球簡易スコアブック（MVP）

アマチュア野球（草野球・少年野球など）向けの、簡易的なスコアブック記録・成績集計Webアプリです。

## 特徴

- **かんたん記録**: 自分のチームの攻撃だけを記録することに特化し、入力の手間を最小限に抑えています。
- **成績自動集計**: 打率や出塁率、防御率などの個人成績を自動で計算します。
- **伝統的スコアシートPDF出力**: 記録した試合を、手書きのようなスコアシート形式でPDF保存できます。
- **モバイルファースト**: ベンチでの操作を想定した、スマートフォンで使いやすいデザイン。

## 技術スタック

- **Framework**: Next.js (App Router)
- **Styling**: Tailwind CSS
- **Local Storage**: 試合データはブラウザのLocalStorageに保存されます。

## セットアップ

まずは依存関係をインストールします：

```bash
npm install
```

開発サーバーを起動します：

```bash
npm run dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開き、アプリを使用できます。

## 使い方

1. 「新しい試合を作成」からチーム名と選手を登録します。
2. 試合画面で、打者ごとに結果（安打、三振、ゴロなど）を選択します。
3. ランナーが進塁した場合は、塁をクリックして進塁・アウトを記録します。
4. 試合が終わったら「終了」ボタンを押し、PDFで記録を保存できます。
